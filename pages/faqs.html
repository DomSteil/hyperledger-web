1. General

What is hyperledger?

Hyperledger is an open source platform for creating private currencies. Anyone can create a ledger, and issue units to it as they wish. The system guarantees that transfers between accounts are always correct (no account can have a negative balance). While control over the issuance in centralised in one party, the system as a whole is decentralised. There are a number of nodes connected to each other in the consensus pool. Every request to one of them is broadcast to all the others. Once a critical number of nodes (more than two thirds) agree that the message is correct, the message is considered committed.

Is hyperledger open source?

Yes, both the hyperledger server and client are completely open source and can be found on github at:

https://github.com/hyperledger/hyperledger-server

https://github.com/hyperledger/hyperledger-cli

Is hyperledger an alt coin?

No, hyperledger is not a fork/clone/derivative of Bitcoin or its derivatives. It is an open source platform for the creation of currencies or assets.

Does hyperledger have its own currency?

No, hyperledger does not have its own designated currency or bridge currency. All currencies in the same hyperledger consensus pool are of equal stature.

Does hyperledger use proof of work or mining?

Is hyperledger a company?

No, hyperledger is the name of the open source project that is not controlled by any single commercial entity.

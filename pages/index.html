Install the Alpha CLI

gem install hyperledger_cli

Create a ledger

> hyperledger new ledger ‘newcoin’ -url ‘newcoin.com’

New ledger created
name: newcoin
url: newcoin.com
id: 4515ecb9b9...
account: 4e0f1fddb5...

Issue ledger

> hyperledger issue ‘newcoin’ 20000

20000 units of newcoin issued
total issued: 20000
in reserve: 20000

Create an Account

> hyperledger new account ‘newcoin’

New account created
ledger: newcoin
id: 0c08722504...
balance: 0

Make a Transfer

> hyperledger transfer 1000 4e0f1fddb5... 0c08722504...

Transfer completed

From
id: 4e0f1fddb5...
balance: 19000

To
id: 0c08722504...
balance: 1000

The Truly Open Payments Protocol

The Hyper Ledger Transaction Protocol (HLTP) is protocol designed from the ground up to enable payments for the internet age. With decentralised nodes running in multiple geographies and jurisdictions, hyperledger provides an amazingly quick, reliable and secure way to record transfers of assets such as digital currencies across the world within seconds, not minutes.

Lightweight - Designed for simplicity and speed
Open - Not controlled by a single company
Equal - No special denominated currency
Fair - No pre-mining or allocations
Efficient - No energy intensive proof of work
Fast - Capable of processing thousands of transactions per second
or checkout our alpha on GitHub

Install the Alpha CLI

gem install hyperledger_cli

Create a ledger

> hyperledger ledger register newcoin https://newcoin.com

Issue Units to a Ledger

> hyperledger issue 20000 newcoin

Create an Account

HLTP achieves consensus across a collection of servers (the consensus pool) via normal web requests without the need for a leader. Clients order their own requests to one or more server nodes which then broadcast all incoming requests to each other with a cryptographic signature as identification.

Once a threshold of verified requests is reached the request is processed. By securely processing requests in this way, the pool of servers will guarantee that the data are always correct. The pool can process requests correctly even if some of the pool are defective or have been hacked.
